# Privacy notice (development draft)

Creator Trust Lens analyses publicly visible Instagram profile, post, and comment information only after the user starts a scan. Analysis is performed locally in the browser.

The initial version:

- does not require a Creator Trust Lens account;
- does not transmit scanned profile information to a Creator Trust Lens server;
- does not sell personal information;
- does not access private Instagram profiles or bypass access controls; and
- stores scan results locally in Chrome storage so the user can review them.

Users can remove locally stored results by clearing the extension's data or uninstalling it.

This document must be reviewed and updated before publication if analytics, payments, crash reporting, cloud sync, or any other external service is introduced.
